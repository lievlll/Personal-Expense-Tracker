# Personal-Expense-Tracker
## Author
Anfalova Yana

## Description
Simple console-based expense tracker built with Python using MVC architecture.

## Features
- Add / delete expenses
- Filter by category
- Calculate total expenses
- Save/load JSON
- Undo system (stack)
- Queue structure included

## Architecture
- MVC pattern
- Services layer
- Data models
- Utilities (Stack, Queue)

## How to run
1.download repository
2.add the repository files to a new folder
3. open the terminal in the folder and start using it

##Tests
python -m unittest tests/test_expenses.py
