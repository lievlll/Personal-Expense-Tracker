# Personal-Expense-Tracker
## Author
Anfalova Yana

## Description
Simple console-based expense tracker built with Python using MVC architecture.

## Features
- Add / delete expenses
- Filter by category
- Calculate total expenses
- Save/load JSON
- Undo system (stack)
- Queue structure included

## Architecture
- MVC pattern
- Services layer
- Data models
- Utilities (Stack, Queue)

## How to run
python main.py
